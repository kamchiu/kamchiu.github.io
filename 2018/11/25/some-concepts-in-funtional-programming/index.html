什么是柯里化函数？

一个函数每次只接收一个参数并且返回另一个接收第二个参数的函数，这个函数又返回一个接收第三个参数的函数，以此类推直到传入了所有的参数。最后的函数处理所有的参数返回结果。假如现在要对两个数进行求和，你可能会这样写：

柯里化和偏函数的区别

偏函数可以接收一个或多个参数，而柯里化将一个接收 n 个参数的函数变成 n 个只有一个参数的函数。所有的柯里化函数都返回偏函数，但不是所有的偏函数都是柯里化函数的返回值。
